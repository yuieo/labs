# Лабораторная работа №7
## Задача
* использование абстрактного базового класса и соотвествующих декораторов для методов,
* иерархия наследования,
* managed - атрибуты,
* минимум 2 dunder-метода у каждого класса.


## Описание 
1. Был взят GUI фреймворк appJar
2. Было: 3 отдельных файла для каждой фигуры (rectangle.py, triangle.py, trapezoid.py)
   
   Стало: Один файл shapes.py со всеми фигурами
   
3. Добавлен базовый класс Shape
   Это Абстрактный базовый класс (ABC), который:
   * Определяет общий интерфейс для всех фигур
   * Гарантирует, что все фигуры будут иметь обязательные методы
   * Позволяет работать с разными фигурами через общий интерфейс

4. Добавлены dunder-методы
   Каждый класс получил:
    * ''__str__'' - для красивого строкового вывода информации о фигуре
    * ''__repr__''- для формального технического представления объекта
    * ''__bool__'' - для проверки ненулевой площади трапеции
  

5. В **main.py** мы работаем с объектами, а не через статические методы. Мы создаем объекты и вызываем их методы.

```
area = Rectangle.area(a, b)
///
shape = Rectangle(a, b)
area = shape.area()
```
