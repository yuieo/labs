## Задание 
* Замыкание-калькулятор, накапливающее результат и поддерживающее 4 арифметичесике операции:
```python
>>> calc = make_calc("*", initial=1)
>>> calc(5)
5
>>> calc(2)
10
```
* Декоратор, который будет запускать функцию указанное число раз с указанными параметрами и возвращать последвательность результатов.

## Как работает

1. make_calc(operation, initial) - создает калькулятор:
   - Хранит результат в замыкании
   - Поддерживает операции: +, -, *, /
   - Начинает с начального значения (initial)

2. repeat(times) - декоратор:
   - Выполняет функцию указанное число раз
   - Возвращает список всех результатов
