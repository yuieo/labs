# Лабораторна работа №5 / Вариант 1 
### Условие задачи
Генератор случайных чисел в заданном диапазоне. Не используйте готовые реализации ГПСЧ. Отфильтруйте вывод генератора, чтобы он не содержал чисел с числом делителей больше n.
К генератору должна быть применена хотя бы одна из функций map, reduce, filter.

### Решение
1. Генерирует псевдослучайные числа в заданном диапазоне.
2. Фильтрует числа, оставляя только те, у которых количество делителей ≤ N.

* Генерация чисел: Используется линейный конгруэнтный метод (LCG).

  ![image](https://github.com/user-attachments/assets/94a84254-b16a-4e76-88fb-154324da04cd)
* Фильтрация: Числа проверяются на кол-во делителей
* Результат: Выводится список чисел, прошедших фильтр.
